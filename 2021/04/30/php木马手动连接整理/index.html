php木马手动连接整理

总所周知，如今我们有很多工具可以来连接上传的木马。但工具显然不是万能的，比如说最近遇到的一道web题，木马通过解析cookie中的反序列化才能成功执行，现有的蚁剑和菜刀并不能满足要求。所以还是有必要整理一下手动通过木马来获取想要的信息。

获取文件名

1.获取某目录下文件名：
//scandir方法
$arr = array();
$data = scandir($path);
foreach ($data as $value){
  if($value != '.' && $value != '..'){
    $arr[] = $value;
  }
}
echo $arr;
2.获取某个目录下所有的目录（包括子目录，子孙目录）和文件，直到最里层：
function searchDir($path,&$files){
  if(is_dir($path)){
    $opendir = opendir($path);
    while ($file = readdir($opendir)){
      if($file != '.' && $file != '..'){
        searchDir($path.'/'.$file, $files);
      }
    }
    closedir($opendir);
  }
  if(!is_dir($path)){
    $files[] = $path;
  }
}
function getDir($dir){
  $files = array();
  searchDir($dir, $files);
  return $files;
}
$filenames = getDir('lss');
foreach ($filenames as $value){
  echo $value.'<br/>';
}

读取文件内容

  • fopen方法：

    readfile("1.txt");

  • file方法：

    $f_arr = file("1.txt");
foreach($f_arr as $cont){
    echo $cont."<br/>";
}

  • file_get_contents方法：

    echo file_get_contents("1.txt");
